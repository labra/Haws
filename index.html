About

HAWS contains a set of libraries and code related with some semantic web projects that I am working on

The code is not production-ready and is mostly proof-of-concept

The source code contains:

  • A purely declarative representation of RDF Graphs using Inductive Triple Graphs
  • Operational semantics of Shape Expressions using a direct translation of the inference rules and a backtracking monad
  • Derivatives implementation of Regular Shape Expressions

Notice that the implementation of both Shape Expressions and Inductive RDF graphs don't contain a parser yet.

By now, the only way to play with them is by declaring the triples/shapes by hand.
